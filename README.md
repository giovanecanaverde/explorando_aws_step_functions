# Resumo de Aprendizado - Fundamentos de Cloud AWS

Este repositório foi desenvolvido com o objetivo de registrar os principais conceitos e aprendizados adquiridos ao longo do Bootcamp GFT - Fundamentos de Cloud com AWS na plataforma DIO. O foco principal do treinamento foi compreender o funcionamento da infraestrutura em nuvem, a integração entre diferentes serviços e a aplicação de boas práticas voltadas para a segurança e o controle de custos em ambientes AWS.

No início da formação, foram abordados os conceitos fundamentais de computação em nuvem, incluindo os modelos de serviço e implantação, além das configurações iniciais de segurança de conta, como a ativação do MFA, e o monitoramento básico de faturamento para evitar gastos imprevistos. Na sequência, o aprendizado avançou para a parte de computação, com foco no entendimento e otimização de instâncias Amazon EC2 e no funcionamento do armazenamento de blocos com o Amazon EBS.

A etapa prática envolveu a criação e manipulação direta de recursos essenciais no console da AWS. Foi realizada a inicialização de uma instância EC2, a configuração de um bucket no Amazon S3 para armazenamento de objetos e o desenvolvimento de uma função básica utilizando o Amazon Lambda, permitindo compreender a dinâmica de arquiteturas serverless.

Na parte de infraestrutura de rede, o treinamento cobriu a estruturação de ambientes isolados por meio de Amazon VPC e subnets, além da configuração de Security Groups para o controle de tráfego. Também foram explorados serviços de distribuição e disponibilidade, como Route 53, CloudFront e Elastic Load Balancer, essenciais para garantir a resiliência e a performance das aplicações.

Por fim, foram estudadas as soluções de banco de dados, diferenciando a aplicação de estruturas relacionais com o Amazon RDS de soluções não-relacionais com o Amazon DynamoDB, bem como a definição de estratégias de backup. Complementando a parte de armazenamento, foi analisado o ciclo de vida de dados entre o Amazon S3 e o Amazon Glacier para otimização de custos, integrando esses conceitos a cenários práticos de desenvolvimento.
